## 权限

- `cloudmist.cs.use` - 允许使用玩家命令（默认：true）
- `cloudmist.cs.admin` - 允许使用管理命令（默认：op）
- `cloudmist.admin` - 允许使用 CS 游戏管理命令（默认：op）

## 配置

游戏设置可通过以下方式配置：

- `/csadmin settings` 命令
- 配置文件修改
- 预设模板使用

可配置项包括：
- 回合时间
- 初始金钱
- 每队最少/最多玩家数
- 胜利所需分数
- 击杀奖励
- 回合胜利/失败奖励
- 冻结时间
- 友军伤害开关

## 安装

1. 下载最新版本的插件 JAR 文件
2. 将文件放入服务器的 `plugins` 文件夹
3. 重启服务器或重载插件
4. 使用管理员命令设置游戏场地
